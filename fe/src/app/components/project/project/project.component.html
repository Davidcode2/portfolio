VR Shooter

Virtual Reality Game

This project was built as part of a uni assignment. It is a simple VR shooter game with a score system. The player can pick up guns and shoot targets. Ricocheting bullets off of surfaces earns the player extra points. The game was developed and tested with the Pico Neo 3 VR headset.
